Support Agents: If there's an emergency, please push the Big Red Button below that's most appropriate given the nature of the emergency.
GENERAL SUPPORT MANAGEMENT ALERT

Click the red button on the left to start a general Support Management alert. If Canvas is down, use the Canvas Down button below. If Cisco is down, use the Cisco Down button below. For anything else, use this button. 

Note that clicking this link will open an email that you must send to actually fire off the alert.
DEFCON-1 BUG ALERT

Click the red button on the left to start a DEFCON-1 Bug alert. This is to alert the management team of a bug about grade things, student work things, access to locked files, privacy things, security things, or academic integrity things. 

Note that clicking this link will open an email that you must send to actually fire off the alert.
CANVAS DOWN ALERT

Click the red button on the left to start a Canvas Down alert. Obviously, use this when Canvas is, you know, down. But also use it if Canvas is slow enough that customers are complaining. 

Note that clicking this link will open an email that you must send to actually fire off the alert.
CISCO DOWN ALERT

Click the red button on the left to start a Cisco Down alert. If Cisco is down or even just slow enough that people are complaining, click this button. 

Note that clicking this link will open an email that you must send to actually fire off the alert.
BRIDGE ISSUE ALERT

Click the red button on the left to start a Bridge Issue alert. If Bridge is having an issue and you are unsure of how to resolve it and no one is available to assist you, click this button. 

Note that clicking this link will open an email that you must send to actually fire off the alert.
